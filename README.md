# SkillBridge Frontend

SkillBridge is a frontend landing page built using **React** and **Vite**.  
The project is designed to help learners understand how a modern learning
platform can bridge the gap between education and industry skills.

This project is **frontend-only** and does not use any backend or database.



## Features
- Responsive Navbar with mobile menu
- Hero section with platform introduction
- Features section with skill-based highlights
- About section explaining mission and vision
- Clean UI with custom CSS
- Beginner-friendly React component structure



## Technologies Used
- React
- Vite
- JavaScript (ES6)
- HTML5
- CSS3
